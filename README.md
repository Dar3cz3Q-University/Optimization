## Setup

### Local development
1. Clone the repository:
   ```shell
   git clone --recursive https://github.com/Dar3cz3Q-University/Optimization.git
   cd Optimization
   ```
2. Run script in [Scripts](/Scripts) directory
3. Open Visual Studio solution or use CMake files
